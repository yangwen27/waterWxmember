## 分销功能
+ 增加商品明细页面，会员可以转发分享
+ 用户通过会员转发分享进入小程序自动绑定为会员的下线，用户下单后会员可以得到佣金。
+ 会员可以在小程序上查看自己的推广下线、下线订单佣金情况，会员可以在小程序上提交提现申请。
## 积分功能
+ 签到积分
+ 购买商品积分
+ 积分兑换礼品
## 抽奖功能
+ 设置活动期间，下单可参与抽奖。
+ 红包（代金券）功能
+ 可以配合软文配发一下红包

## 水站端
+ 水站端（可在公众号接受订单信息）
+ 水站申请入驻，提交资料后台审核通过后可以接单
+ 水站可以在小程序上设置配送范围
+ 水站可设置营业状态：正常营业，休息中
+ 订单统计（能区分水票订单、品牌及数量）
+ 累计收款统计
+ 桶押金记录、桶押金核销（客户申请退桶由所收押金水站处理）
+ 商品上下架功能（商品是后台统一上线，水站短只能选择）
+ 添加送水工，水站管理员分享链接给送水工，送水工绑定微信
